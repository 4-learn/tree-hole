# 樹洞

有一個概念叫「樹洞」，通常用來比喻人們傾訴心事的對象或地方。這個概念來源於傳說或文學作品，人們會對著樹洞說出自己心裡的秘密，像是將自己的煩惱或心事藏在那裡，樹洞則象徵一個安全的空間，不會把秘密洩露出去。
